## 1. Что это за приложение?
Это сайт, где можно создавать события (дни рождения, встречи, дедлайны) и приглашать друзей.

**Пример использования:**
1. Иван создаёт событие "День рождения"
2. Указывает дату: 25 марта, время: 18:00, место: Кафе "Уют"
3. Приглашает друзей: Мария, Пётр, Анна
4. Все получают уведомления

## 2. Что может делать приложение?

### Для пользователя:
1. **Зарегистрироваться** - создать аккаунт
2. **Войти** - в свой аккаунт
3. **Создать событие** - добавить новое мероприятие
4. **Посмотреть календарь** - увидеть все события
5. **Пригласить друзей** - отправить приглашения
6. **Получить напоминание** - перед событием

### Технические требования:
- Работает в браузере Chrome
- Загружается за 2-3 секунды
- Сохраняет данные
